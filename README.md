# Relatório de Tráfego

Aplicação Flask para gerar relatórios a partir de CSVs de campanhas.

## Requisitos
- Python 3.x
- pip

## Instalação
```
pip install -r requirements.txt
```

## Execução
- Via Python:
```
python app.py
```
- Ou via script:
```
iniciar_sistema_web.bat
```

## Uso
- Acesse http://127.0.0.1:5000
- Faça upload do CSV
- O relatório é gerado na página

## Observações
- A pasta `uploads/` não deve ser versionada.
